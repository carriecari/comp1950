Homework 1: Publish your COMP1950 homepage

Instruction

Assigned: Session 01 Due: Session 02
Specifications: Create a homepage to serve as the hub for all your work in COMP1950. As subsequent homework assignments are issued, you will add links to them on this page. Your homepage should state your full name, or nickname if you prefer. Include some biographical information about yourself (such as from your introduction today) and a photo. Also include:
  • A brief overview of your skill level, web development work you have done so far (if any)
  • A list of tools, plug-ins, applications or IDEs that you use for web development,including links to the web pages of these tools.
  • Links to some of your favorite sites, accompanied by brief descriptions explaining why these sites are favorites
Use your understanding of HTML and CSS to the limit of your knowledge, but ensure that it is usable, presentable and coherent. Use CSS, not HTML tables, to define the layout of this page. Include the use of both .class and #id CSS selectors. Do NOT use any Javascripting or server side scripting. Be prepared to share this page with your colleagues in subsequent homework assignments: design and build a page you can be proud of. This page will be used as the entry point for all of your COMP1950 work, edited each week. Don't make your instructor have to search for your homework.
Evaluations: 10 marks for meeting all specifications.
Notes: This and all subsequent COMP 1950 homework assignments are due the day before thefollowing class unless otherwise noted (eg: if assigned Monday, Jan 1, homework is dueTuesday, Jan 2 11:59pm). Only files located on the COMP 1950 web server will bemarked. All submissions must include:
  • Well formed, valid, semantic HTML. Structural errors are not permitted (eg: misplacing a li tag, or duplicating an id assignment). Non-structural errors are permissible (eg: using a color name not recognized by the validator)
  • Include a referer link on all your COMP1950 pages
  • A code comment with your full name
  • Good file/folder naming conventions
  • Good code practices. (Comment major sections of code, use tabs to indicate tag inheritance)

Rubrics

Dropbox

Your first homework will be uploaded to your S01 folder.

See an example at Lynda's Demostudent Index Page.

Submit your work by sending a text file (call it S01homework.txt) containing:

  1. the URL to your myindex.html page on the http://www.bcitcomp.ca/ server
  2. A paragraph describing your design (e.g. "Two column layout with menu floating left ... ") 
  3. A statement discussing any validation failures you did not fix up. Why you left them. If you couldn't fix them what you tried to do so, etc.

Upload the text file to the dropbox.

Also provide the URL to the homework assignment in the comments for the submission and make the link clickable.

A detailed explanation of the homework can be found on the lead instructor's site at: http://www.bcitcomp.ca/_shared_resources_1950/downloads/comp1950homework01.pdf
